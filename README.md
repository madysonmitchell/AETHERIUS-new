# AETHERIUS
### A Quadrinary Biomimetic Computing Architecture

AETHERIUS is a novel computing framework that replaces binary logic with a 
four-state system inspired by mycelial networks. The system operates through 
controlled gallium-alloy fluid dynamics in microfluidic channels, using 
magnetic field gradients for computation and memory storage.
